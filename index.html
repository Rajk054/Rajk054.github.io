Transaction Fraud Detection

Compares an autoencoder, a focal-loss MLP, and a GraphSAGE GNN on PaySim transactions — testing whether a transaction graph catches coordinated fraud that row-by-row models miss. Chronological split, PR-AUC ranking, cost-based evaluation.

Python PyTorch GraphSAGE PaySim

Sequential Recommendation (SASRec)

Benchmarks matrix factorization, a two-tower retrieval model, and a SASRec Transformer on MovieLens 1M with full-catalog NDCG/Recall. Includes a FastAPI + FAISS HNSW service comparing exact vs. approximate retrieval.

Python PyTorch FAISS FastAPI
Vision / Research soon
DINO vs. ImageNet for Detection

Asks whether frozen self-supervised DINO features transfer better than supervised ImageNet features for low-label object detection. The same from-scratch YOLO-style head is attached to both frozen backbones on Pascal VOC, measured by mAP@0.5.

Python PyTorch DINO ViT

Proximal Policy Optimization implemented directly in PyTorch and trained on Atari Breakout. The clipped objective, GAE, rollout buffer, entropy regularization, and stability diagnostics are all local code; Gymnasium only runs the environment.

Capstone Research

Predictive scene-change detection

The question I'm working on

Can a lightweight predictive model trained over DINOv2 features from a persistent 3D Gaussian Splatting scene memory produce a calibrated latent error signal that detects object-level scene changes from sparse post-change observations — and does this predictive error outperform direct feature comparison and 3DGS-based change-detection baselines?
